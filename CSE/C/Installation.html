Installation

Windows Setup

To compile C programs on Windows, we need to install the GCC compiler. The easiest way to get a modern 64-bit compiler is by using MSYS2.

Step 1: Download and Install MSYS2

  • Go to the official website: https://www.msys2.org
  • Download: Download the installer executable (usually named something like msys2-x86_64-*.exe).
  • Install: Run the executable and follow the installation wizard, keeping the default settings.

Step 2: Install the GCC Toolchain

Installing MSYS2 alone does not install the compiler. You must download it through the MSYS2 terminal.

  • Open MSYS2: Launch the "MSYS2 UCRT64" terminal from your Start menu.
  • Run the installation command: Type the following command and press Enter. Type 'Y' when prompted to confirm.
pacman -S mingw-w64-ucrt-x86_64-gcc

Step 3: Add MSYS2 to Environment Variables

You need to add the compiler's path to your system so you can run it from any command prompt.

  • Open Environment Variables: In the Windows search bar, type "Environment Variables" and click on "Edit the system environment variables".
  • Edit the Path: Click the "Environment Variables" button. Under "System variables" (or "User variables"), find the "Path" variable and click "Edit".
  • Add the new path: Click "New" and paste the MSYS2 UCRT64 bin path. If you installed it in the default location, it will be: C:\msys64\ucrt64\bin. Click OK to save.

Step 4: Verify Installation

To check if the compiler is set up correctly, open a new Command Prompt or PowerShell and run:

gcc --version
  • Success: If the command returns version information for GCC, your installation was successful.
  • Error: If you see an error saying 'gcc' is not recognized as an internal or external command, your Environment Variables are not set correctly. Restart your terminal or double-check the path.

Linux Setup

Most Linux distributions provide excellent C language support by default. You can easily download the GCC compiler using your distribution's package manager.

Ubuntu / Debian

Run the following commands in your terminal to update your package lists and install the build-essential package, which includes GCC and other necessary compiling tools.

sudo apt update
sudo apt install build-essential

Fedora / RHEL

sudo dnf groupinstall "Development Tools"

macOS Setup

macOS comes with its own set of development tools provided by Apple, which includes a C compiler (Apple Clang, which handles GCC commands).

Step 1: Install Command Line Tools

You don't need to install the full Xcode application. You can install just the command-line tools by opening your Terminal and running:

xcode-select --install

Step 2: Complete the Prompt

  • A software update popup window will appear asking if you want to install the tools.
  • Click Install and agree to the terms of service.
  • Once the download and installation finish, you can verify it by typing gcc --version in the terminal.
